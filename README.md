# Soujanya Makeovers - Professional Makeup Artist Website

A modern, responsive website for Soujanya Makeovers, a professional makeup artist specializing in bridal makeup, hairstyling, and saree draping services.

## Features

- Fully responsive design that works on all devices
- Modern and clean user interface
- Service showcase with icons
- Portfolio gallery
- Testimonial slider
- Contact form for appointment booking
- Smooth scrolling navigation
- Animated elements on scroll
- Mobile-friendly navigation menu

## Technologies Used

- HTML5
- CSS3 (with CSS Variables for easy theming)
- JavaScript (Vanilla JS, no dependencies)
- Font Awesome Icons
- Google Fonts (Poppins & Playfair Display)

## Getting Started

### Prerequisites

- A modern web browser (Chrome, Firefox, Safari, Edge)
- A code editor (VS Code, Sublime Text, etc.)

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/souji-makeover.git
   ```

2. Navigate to the project directory:
   ```bash
   cd souji-makeover
   ```

3. Open `index.html` in your browser to view the website locally.

## Customization

### Changing Content

1. **Home Page**: Edit the content in `index.html` to update text, images, and links.
2. **Services**: Update the services section in `index.html` to reflect your specific offerings.
3. **Portfolio**: Add your work samples by updating the `portfolioItems` array in `js/script.js`.
4. **Testimonials**: Update the testimonials in `index.html`.
5. **Contact Information**: Update the contact details in the contact section of `index.html`.

### Styling

- Colors and fonts can be customized in the `:root` section of `css/style.css`.
- Layout adjustments can be made in the respective sections of the CSS file.

### Images

Replace the placeholder images in the `images/` directory with your own high-quality images. Use `images/hero-bg.jpg` for the hero background and `images/portfolio/*.jpg` for portfolio items. Make sure to maintain the same file names or update the references in the HTML and JavaScript files.
